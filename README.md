# 🏭 기업 ERP 프로젝트 (Front-end)
#### Back-end 작업물은 [이곳](https://github.com/aid95/expressjs-erp-project)에서 확인할 수 있습니다.

## 사용 기술

- React.js
- React hooks
- Apollo client
- styled-components

## 실행 환경

- Node.js

## 프로젝트 실행

```.env```은 루트 디렉토리에 위치합니다.
```shell
PORT=3000
```

프로젝트 실행은 아래의 명령어를 수행합니다.
```shell
# RUN SERVER
yarn start
```
